# DiamanteDeXAP
## Descrição
O Programa pega um valor inteiro ímpar e o usa como a base para criação de uma estrutura em formato de diamante.
## Funcionalidade
Após receber um número ímpar inteiro, o programa cria uma linha principal composta pelo caractere X do tamanho informado, diminuindo de tamanho em 2 unidades, até chegar no número minimo 1.
